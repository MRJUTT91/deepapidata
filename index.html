API Integration Information

To connect your C# application to this API, use the following endpoints:

POST /api/message - Send a message from C# app to website
GET /api/messages - Retrieve all messages

Example C# code for sending a message:

var client = new HttpClient();
var response = await client.PostAsJsonAsync("https://mrjutt91.github.io/deepapidata//api/message", new {
  user = "C# App",
  text = "Hello from C#"
});
